# Non-Goals: academyinfo-mcp v0.1

v0.1 will not scrape websites.

v0.1 will not call live OpenAPI endpoints.

v0.1 will not require API keys.

v0.1 will not require `DATA_GO_KR_SERVICE_KEY`.

v0.1 will not require `ACADEMYINFO_SERVICE_KEY`.

v0.1 will not redistribute dataset `15139279`.

v0.1 will not bundle raw, normalized, seed, sample, or fixture data from `15139279`.

v0.1 will not use dataset `15139279` for default `employment_rate`.

v0.1 will not provide granular, per-department, or health-insurance-linked
employment statistics.

v0.1 will not produce official rankings.

v0.1 will not claim official endorsement.

v0.1 will not correct official statistics.

v0.1 will not infer missing units or columns.

v0.1 will not invent API endpoints, operation names, request parameters, response fields, file columns, or units.

v0.1 will not mutate raw source files.

v0.1 will not discard raw rows needed for auditability.

v0.1 will not mix code license and data license obligations.

v0.1 will not expose API keys, service keys, credentials, private paths, local user names, or machine identifiers in logs, MCP responses, manifests, documentation examples, package artifacts, snapshots, or error messages.

v0.1 will not write logs to stdout in stdio MCP mode.
